# 🏷️ Sticker Label Printing System

A simple and modern **Sticker Label Printing System** built with **Java (JDK 17)** and developed using **NetBeans**.  
This application is designed for shops and businesses that need to print product labels quickly and cleanly with essential details such as:

- Product / item name
- Price
- Expiry / date fields
- Shop name
- Address
- Contact number
- Quantity-based printing

---

## ✨ Features

- Clean and easy-to-use Java desktop application
- Label / sticker printing support
- Add item name, price, date details, and quantity
- Print shop name, address, and contact details on every sticker
- Supports batch printing by quantity
- Built using **JDK 17**
- Developed in **NetBeans**
- Simple, modern, and practical for small shops

---

## 📦 Requirements

Before running the project, make sure you have:

- JDK 17 installed
- A compatible printer for sticker output
